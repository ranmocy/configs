# Ranmocy's configs

Bundle of all useful configs

## Usage

```bash
git clone https://github.com/ranmocy/configs.git ~/.oh-my-configs && bash ~/.oh-my-configs/install.sh
```

To upgrade all packages:

```bash
bash ~/.oh-my-configs/upgrade.sh
```
